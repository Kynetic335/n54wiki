# Fuel Scalar

> **Source confidence:** Needs validation

> Educational reference only. This page does not publish tune values.

## What this map controls

A fuel scalar adjusts overall fueling to match injector/fuel characteristics so commanded mixture matches actual.

## Why it matters

Incorrect scaling makes lambda targets miss and trims grow. It must match the actual injectors and fuel.

## Related log channels

- Lambda/AFR
- STFT/LTFT
- Fuel rail pressure

## What changes are visible in logs

Changes show as shifted fuel trims and measured mixture relative to target.

## What can go wrong if changed blindly

- Lean/rich offset across the range
- Large fuel trims
- Drivability/idle issues

## Mechanical problems that can look like a tuning problem

- Wrong injector coding
- Leaking injectors
- Wrong ethanol content

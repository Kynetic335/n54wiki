# N54 Public Safety No Private Files

Keep the public wiki separate from private calibration assets.

## Hard Prohibitions

- Do not expose raw private BIN, XDF, DAMOS, A2L, OLS, tune, or customer files.
- Do not summarize private files into publishable content.
- Do not add file upload, file browser, download, patch, or generation flows for ECU binaries or private tune material.
- Do not add flashing, BIN writing, checksum correction, tune generation, or automatic calibration patching.
- Do not create examples from protected directories or private inventories.

## Safe Alternatives

- Provide conceptual explanations.
- Use synthetic, clearly labeled examples that contain no private values.
- Reference approved public content already intended for the wiki.
- Offer an exposure-risk review when the task is about preventing leaks.

## If A Request Crosses The Boundary

Refuse the unsafe/private part briefly. Continue with a safe public educational explanation when possible.

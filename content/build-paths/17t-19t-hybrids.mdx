# 17T / 19T Hybrids

> **Source confidence:** Needs validation

## What it is

A hybrid-turbo build upgrades the stock-location twins (commonly 17T/19T) for more airflow while keeping the twin-turbo layout.

## Why it matters

Hybrids raise the airflow ceiling beyond stock with relatively quick spool — but need matching fueling, cooling, and boost control.

## How it works

A build path is a coherent combination of fuel, hardware, and a matching calibration. This wiki describes the concept and prerequisites; it does not publish tune values.

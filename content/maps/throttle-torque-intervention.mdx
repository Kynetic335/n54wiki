# Throttle Torque Intervention

> **Source confidence:** Needs validation

> Educational reference only. This page does not publish tune values.

## What this map controls

This logic governs when and how the DME closes the throttle to intervene on torque (protection, limiting, smoothing).

## Why it matters

It directly causes throttle closures seen in logs. Misunderstanding it leads to chasing the wrong cause for power cuts.

## Related log channels

- Throttle angle
- Pedal position
- Torque request
- Requested vs actual load

## What changes are visible in logs

Changes show as different throttle-closure behavior relative to torque demand.

## What can go wrong if changed blindly

- Unexpected power cuts
- Drivability issues
- Masking the real limiter

## Mechanical problems that can look like a tuning problem

- Throttle body faults
- Pedal sensor faults

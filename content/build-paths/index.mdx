# Build Paths

The Build Paths section describes coherent N54 build directions — from stock through stock-turbo ethanol builds, hybrids, and single turbo. Each path starts from maintenance and supporting parts before discussing any tuning concept. This wiki explains prerequisites and trade-offs; it does not publish tune values or flashing steps.

- **[Full E85](/wiki/build-paths/full-e85)** — High-ethanol build.
- **[17T / 19T Hybrids](/wiki/build-paths/17t-19t-hybrids)** — Upgraded twin-scroll hybrids build.

- **[Single Turbo](/wiki/build-paths/single-turbo)** — Top-tier airflow build.
- **[Port Injection Build](/wiki/build-paths/port-injection)** — Adding PI for fuel headroom.

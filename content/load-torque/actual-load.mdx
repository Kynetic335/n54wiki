# Actual Load

> **Source confidence:** Needs validation

## What it is

Actual load is the realized load the engine achieves, derived from measured airflow/manifold pressure.

## Why it matters

Actual falling below requested under load points at a boost shortfall (often mechanical) or an active limiter.

## How it works

The DME measures the achieved load and compares it to the request to manage boost and fueling.

## Common symptoms or failure modes

- Actual below requested under load

## What to check in logs

- Actual load
- Requested load
- Boost target vs actual

## Mechanical checks first

Smoke-test for boost leaks when actual load lags requested.

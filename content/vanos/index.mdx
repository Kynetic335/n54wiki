# VANOS

VANOS is BMW's variable camshaft timing system. On the N54 it adjusts intake and exhaust cam timing to improve spool, power, and efficiency across the rev range. This section covers how it works, its solenoids, common faults, service, and the tuning tables that command cam timing.

# Load Target Map

> **Source confidence:** Needs validation

> Educational reference only. This page does not publish tune values.

## What this map controls

The load target map sets the load the DME aims to achieve across rpm and conditions. It is upstream of boost and a central tuning map.

## Why it matters

Load target shapes how much air/torque the engine requests. Raising it without supporting fuel, boost control, and limiters causes problems.

## Related log channels

- Requested load
- Actual load
- Boost target vs actual

## What changes are visible in logs

Changes appear as a different requested-load curve and corresponding boost/torque behavior.

## What can go wrong if changed blindly

- Requesting more than fuel/boost can support
- Hitting limiters or protection
- Lean or knock if unsupported

## Mechanical problems that can look like a tuning problem

- Boost leaks
- Fuel-supply limits
- Worn turbos

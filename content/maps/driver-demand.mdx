# Driver Demand Map

> **Source confidence:** Needs validation

> Educational reference only. This page does not publish tune values.

## What this map controls

The driver demand map converts pedal position (and rpm) into a torque request, shaping throttle response and feel.

## Why it matters

It changes how the pedal feels without changing peak power. Overly aggressive maps can feel twitchy; lazy maps feel unresponsive.

## Related log channels

- Pedal position
- Throttle angle
- Torque request

## What changes are visible in logs

Changes show as a different relationship between pedal input and commanded torque/throttle.

## What can go wrong if changed blindly

- Twitchy or unpredictable response
- Drivability complaints

## Mechanical problems that can look like a tuning problem

- Throttle body or pedal sensor faults

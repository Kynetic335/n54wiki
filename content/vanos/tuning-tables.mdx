# VANOS Tuning Tables

> **Source confidence:** Needs validation

> Educational reference only. This page does not publish tune values.

## What this map controls

The VANOS tuning tables command target intake and exhaust cam angles across rpm and load. They influence spool and power delivery.

## Why it matters

Cam timing changes affect spool, torque shape, and emissions. Changing them blindly can hurt drivability or trigger faults.

## Related log channels

- VANOS intake target vs actual
- VANOS exhaust target vs actual
- Boost target vs actual (spool context)

## What changes are visible in logs

Changes appear as different commanded cam angles and altered spool/torque behavior in logs.

## What can go wrong if changed blindly

- Rough running or stalls
- VANOS deviation faults
- Worse spool if mis-set

## Mechanical problems that can look like a tuning problem

- Dirty/failed VANOS solenoids
- Low or dirty oil
- Mechanical VANOS wear

# Rev Limiters

> **Source confidence:** Needs validation

> Educational reference only. This page does not publish tune values.

## What this map controls

Rev limiter maps set the maximum engine rpm and how the limit is enforced (fuel/spark cut).

## Why it matters

Raising the rev limit stresses valvetrain and rotating assembly. It should respect the engine's mechanical limits.

## Related log channels

- RPM
- Misfire counters (cut behavior)

## What changes are visible in logs

Changes show as a different rpm ceiling and cut behavior.

## What can go wrong if changed blindly

- Valvetrain/mechanical stress
- Reliability loss beyond safe rpm

## Mechanical problems that can look like a tuning problem

- Misfires near redline that mimic a limiter

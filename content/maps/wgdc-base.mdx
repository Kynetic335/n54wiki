# WGDC Base Map

> **Source confidence:** Needs validation

> Educational reference only. This page does not publish tune values.

## What this map controls

The WGDC base map sets the baseline wastegate duty cycle the boost-control system starts from before corrections.

## Why it matters

A poor base makes the controller work harder and can cause overshoot or slow response. It interacts tightly with the correction map.

## Related log channels

- WGDC
- Boost target vs actual

## What changes are visible in logs

Changes appear as a different baseline WGDC and altered spool/overshoot behavior.

## What can go wrong if changed blindly

- Boost overshoot
- Oscillation
- Underboost if too low

## Mechanical problems that can look like a tuning problem

- Vacuum leaks
- Failing solenoids
- Worn wastegates

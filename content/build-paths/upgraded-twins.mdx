# Upgraded Twins

> **Source confidence:** Needs validation

## What it is

An upgraded-twins build uses larger twin turbos for more power than stock or basic hybrids, retaining the twin layout.

## Why it matters

More airflow needs proportionally more fuel and cooling; supporting hardware and maintenance must scale with the target.

## How it works

A build path is a coherent combination of fuel, hardware, and a matching calibration. This wiki describes the concept and prerequisites; it does not publish tune values.

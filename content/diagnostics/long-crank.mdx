# Long Crank

> **Source confidence:** High confidence (behavior). Pressure values — verify by VIN/ROM ID.

## What the symptom means

Long crank is when the engine turns over noticeably longer than normal before it catches and starts. To fire promptly, the direct-injection N54 needs fuel rail pressure built quickly at startup. When pressure builds slowly or low, cranking drags out. On the N54 this is a classic early warning of a **failing HPFP**, though it is not the only cause.

## Common causes

- **Failing / worn HPFP** — slow to build rail pressure at start (most common).
- **Weak LPFP** — not feeding the HPFP fast enough.
- **Leak-down** in the fuel system (pressure bleeds off when parked).
- **Battery / starter weakness** — slow cranking speed.
- **Sensor or ignition issues** affecting start.

## What to log

- [HPFP / rail pressure](/wiki/logging/hpfp-pressure) at start (build speed and value)
- [LPFP pressure](/wiki/logging/lpfp-pressure) at start
- Battery voltage / cranking behavior if available

## What a healthy pattern looks like

- Rail pressure builds **quickly** at crank and reaches its start target.
- LPFP supplies steady low-side pressure.
- Engine catches promptly.

## What a bad pattern can suggest

- Rail pressure **building slowly or staying low** at crank → HPFP weakness ([HPFP Failure](/wiki/diagnostics/hpfp-failure)).
- Low-side sag at start → LPFP issue ([LPFP Failure](/wiki/diagnostics/lpfp-failure)).
- Pressure that bled off while parked → leak-down.
- Slow cranking with normal fuel pressure → battery/starter.

## Mechanical checks first

1. Test **HPFP** behavior at start (rail pressure build).
2. Test **LPFP / low-side** supply.
3. Check **battery and starter**.
4. Scan for stored **fuel-pressure codes** (e.g. [29DC](/wiki/diagnostics/29dc-fuel-pressure)).

## Tuning-related causes second

Long crank is a hardware/fuel-supply symptom, not a tuning value to dial around. Do not mask it in software. If a pump is failing, replace it.

## When to stop tuning

If long crank appears with low/slow rail pressure, address the fuel system before any tuning. A car that barely starts is not a safe tuning baseline.

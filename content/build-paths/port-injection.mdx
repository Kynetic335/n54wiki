# Port Injection Build

> **Source confidence:** Needs validation

## What it is

A port-injection build adds intake-port injectors to supplement the factory direct injection on high-power/high-ethanol setups.

## Why it matters

When DI runs out of fuel capacity, PI adds headroom and can help keep valves cleaner. It is a hardware addition with its own control needs.

## How it works

A build path is a coherent combination of fuel, hardware, and a matching calibration. This wiki describes the concept and prerequisites; it does not publish tune values.

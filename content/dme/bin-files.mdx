# BIN Files

> **Source confidence:** Needs validation

## What it is

A BIN file is a binary image of the DME calibration/ROM. It contains the maps the engine runs. This wiki explains the concept only and does not host or distribute private BIN files.

## Why it matters

Understanding what a BIN is helps explain ROM matching, checksums, and definitions — without ever publishing private calibration data.

## How it works

The BIN is a raw image. Definitions (XDFs) describe where maps live inside it. Checksums validate integrity.

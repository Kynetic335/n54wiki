# Boost Target Map

> **Source confidence:** Needs validation

> Educational reference only. This page does not publish tune values.

## What this map controls

The boost target map sets the boost pressure the DME aims for. WGDC works to achieve it; sensors confirm actual.

## Why it matters

Boost is the most-touched tuning value and the easiest to over-request. The target must match turbo capacity, fueling, and cooling.

## Related log channels

- Boost target
- Boost actual
- WGDC
- IAT

## What changes are visible in logs

A higher target shows as higher commanded boost and increased WGDC effort.

## What can go wrong if changed blindly

- Overboost/limp mode
- Exceeding turbo/fuel capacity
- Knock from too much boost for the fuel

## Mechanical problems that can look like a tuning problem

- Boost leaks (look like underboost)
- Worn wastegates
- Vacuum leaks

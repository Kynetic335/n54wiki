# VANOS Exhaust Map

> **Source confidence:** Needs validation

> Educational reference only. This page does not publish tune values.

## What this map controls

The VANOS exhaust map commands target exhaust cam angles, strongly influencing turbo spool.

## Why it matters

Exhaust cam timing is a key spool lever. Mis-set values hurt spool or trigger faults.

## Related log channels

- Exhaust VANOS target vs actual
- Boost target vs actual

## What changes are visible in logs

Changes show as different commanded exhaust cam angles and spool behavior.

## What can go wrong if changed blindly

- Worse spool
- VANOS deviation faults
- Rough running

## Mechanical problems that can look like a tuning problem

- Dirty/failed solenoids
- Low/dirty oil
- Mechanical wear

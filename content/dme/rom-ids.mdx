# ROM IDs

> **Source confidence:** Needs validation

## What it is

A ROM ID identifies the specific calibration image a DME is running. Common N54 ROM IDs are referenced as short codes (for example IJE0S, I8A0S, INA0S, IKM0S).

## Why it matters

Every map definition is tied to a specific ROM. The ROM ID is how you confirm a definition matches before reading or interpreting any map.

## How it works

The ROM ID is read from the DME with diagnostic software. It corresponds to a particular factory calibration baseline.

## What to check in logs

- ROM ID field via diagnostic software

## Mechanical checks first

Always confirm ROM ID first. Mismatched definitions produce wrong addresses and bad data. Verify by ROM ID.

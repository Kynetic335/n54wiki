# Injector Coding

> **Source confidence:** Needs validation

## What it is

Injector coding is the process of telling the DME each injector's index/flow data so it can fuel accurately. This wiki explains the concept, not a flashing procedure.

## Why it matters

Without correct coding, fueling is off and idle/running quality suffers. It is a required step after injector replacement.

## How it works

Diagnostic software writes the injector index values to the DME so compensation matches the installed injectors.

## Common symptoms or failure modes

- Rough running after injector replacement if coding skipped or wrong

## What to check in logs

- Fuel trims
- Misfire counters

## Mechanical checks first

Verify coding matches the physically installed injectors.
